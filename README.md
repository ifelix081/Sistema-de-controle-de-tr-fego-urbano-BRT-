# Sistema-de-controle-de-tr-fego-urbano-BRT-
Sistema de Controle de Tráfego Urbano (BRT) é uma solução tecnológica que monitora e otimiza o fluxo de ônibus em corredores exclusivos. Utiliza GPS, sensores e dados em tempo real para reduzir atrasos, melhorar a mobilidade e fornecer previsões aos passageiros por aplicativos e painéis.
